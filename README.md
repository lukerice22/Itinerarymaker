# 🗺️ Itinerarymaker

**Itinerarymaker** is a lightweight web application that takes the guesswork out of travel planning. Whether you're heading to a new city for a weekend or a week, this tool generates a structured daily schedule based on your destination and stay duration.

## 🚀 Features

* **Smart Scheduling:** Generates tailored itineraries based on the number of days you'll be visiting.
* **Location-Specific:** Provides suggestions focused on the unique highlights of your chosen destination.
* **Minimalist Design:** A clean, easy-to-use interface for quick planning on the go.
* **Instant Results:** Get your full trip breakdown in seconds.

## 🛠️ How it Works

1.  **Enter Location:** Input the city or country you are visiting.
2.  **Set Duration:** Choose how many days your trip will last.
3.  **Generate:** View your personalized daily itinerary.

## 💻 Technical Setup

### Local Development

To run this website locally on your machine:

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/lukerice22/Itinerarymaker.git](https://github.com/lukerice22/Itinerarymaker.git)
    ```
2.  **Open the project folder**
    ```bash
    cd Itinerarymaker
    ```
3.  **Launch the site**
    * If it's a static site, simply open `index.html` in your browser.
    * If using a local server (like Live Server in VS Code), click "Go Live."
